iTunes Lyrics
=============
by tranquility, with portions from lLyrics for Rhythmbox, https://github.com/dmo60/lLyrics

Requires
--------

 - Python
 - osascript
 - urllib2

Running
-------

To run the program, open iTunes and play a song, then type in your terminal
 python lyrics.py|less
